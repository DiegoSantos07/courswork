In this project I was responsible for the following tasks:

Splash Screen

I was in charge of creating the Splash Screen for the group project which leads to the Home page after loading for 4 seconds. I created the loading animation, the motto for the website mission, and put down the names of the group members beneath it.



Shop

As Student 1, I have created the website's Shop. Here I have listed a few items that may interest the user such as recyclable t-shirts and reusable water bottles. The user is free to add the items to the cart and change the quantity of the items it wishes to buy. It can then fill out the Checkout form, prompting the user to fill all the mandatory fields, otherwise the program will not allow them to checkout. If the user fills out all the required fields the program then responds with the confirmation message.



Content page

I have created a content page on what Climate Action is and what the goals are, as well as things to do in order to avoid harming the planet.
